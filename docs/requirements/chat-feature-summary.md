# チャット機能 - ユーザー向け説明資料

## 📱 概要

備品管理システムに**リアルタイムチャット機能**を追加し、ユーザーと管理者が備品について気軽に問い合わせ・相談できるようにします。

---

## 🎯 何ができるようになるの？

### 1. 備品について問い合わせ
備品詳細ページから、その備品について直接管理者に質問できます。

**例**:
- 「このノートPCの使い方を教えてください」
- 「プロジェクターが故障しているようです」
- 「明日この備品を借りられますか？」

### 2. 一般的な問い合わせ
ヘッダーのチャットアイコンから、備品に関係ない質問もできます。

**例**:
- 「新しい備品を追加してほしい」
- 「貸出期限を延長できますか？」
- 「システムの使い方がわかりません」

### 3. 画像を添付して状況を共有
テキストだけでなく、写真を送ることで状況を正確に伝えられます。

**例**:
- 故障した備品の写真
- エラー画面のスクリーンショット
- 備品の状態確認

---

## 💬 使い方

### チャットを始める（2つの方法）

#### 方法1: 備品詳細ページから
```
1. 備品詳細ページを開く
2. 「問い合わせ」ボタンをクリック
3. チャット画面が開きます
```

#### 方法2: ヘッダーのチャットアイコンから
```
1. 画面上部のチャットアイコン💬をクリック
2. 「新規チャット」ボタンをクリック
3. チャット画面が開きます
```

### メッセージを送る

```
1. 画面下のメッセージ入力欄に文字を入力
2. Enterキーを押す（または「送信」ボタンをクリック）
3. すぐに管理者に届きます！
```

### 画像を送る

```
1. メッセージ入力欄の左にある📎ボタンをクリック
2. 送りたい画像を選択
3. プレビューが表示されるので確認
4. 必要に応じてメッセージを追加
5. 送信ボタンをクリック
```

## ✨ 便利な機能

### 📬 未読メッセージ通知
新しいメッセージが届くと、ヘッダーのチャットアイコンに **赤いバッジ** が表示されます。

```
💬(2) ← 未読メッセージが2件あります
```

### 💬 リアルタイム返信
管理者が返信すると、**すぐに画面に表示**されます。ページを更新する必要はありません。

### 📜 履歴が残る
過去30日間のやり取りは保存されるので、後から見返すことができます。

### 🔍 チャット一覧
複数の問い合わせをしている場合、一覧で確認できます。未読件数や最新メッセージも表示されます。

## ⚠️ 注意事項

### 📝 メッセージについて
- **文字数制限**: 1回のメッセージは1000文字まで
- **履歴保存**: メッセージは30日間保存され、その後自動削除されます
- **編集・削除**: 送信後のメッセージは編集・削除できません

### 🖼️ 画像について
- **対応形式**: PNG、JPG、GIF形式のみ
- **ファイルサイズ**: 1ファイル5MBまで
- **枚数制限**: 1回のメッセージにつき1枚まで
- **保存期間**: 画像も30日後に自動削除されます

### 📱 動作環境
- **ブラウザ**: Chrome、Firefox、Safari、Edge（最新版）
- **デバイス**: パソコン、タブレット、スマートフォン
- **インターネット接続**: リアルタイムチャットのため、常時接続が必要

## ❓ よくある質問

**Q: チャットは誰が見られますか？**
A: あなたと管理者のみが見られます。他のユーザーには見えません。

**Q: 緊急の場合でも大丈夫ですか？**
A: リアルタイムで届きますが、緊急の場合は電話などの直接連絡も併用してください。

**Q: 夜間や休日でも使えますか？**
A: メッセージは24時間送信できますが、管理者の返信は営業時間内になります。

**Q: 古いメッセージを見返すことはできますか？**
A: 過去30日間のメッセージは履歴から確認できます。

**Q: スマートフォンでも使えますか？**
A: はい、スマートフォンでも快適に使えるよう最適化されています。

**Q: 間違えてメッセージを送ってしまいました**
A: 送信後の削除・編集はできませんので、追加でメッセージを送って訂正してください。
